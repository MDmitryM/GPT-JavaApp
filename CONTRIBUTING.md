Contributing Guidelines
Welcome to our open source Java application that uses the ChatGPT API! We appreciate your interest and your willingness to contribute.

Project Overview
Our application is built on Java 17.0.2 and uses the ChatGPT API to provide an AI-powered chatbot interface. The chatbot can engage in conversations with users and provide helpful responses.

Getting Started
To contribute to the project, please follow these steps:

Fork the repository to your own GitHub account.
Clone the repository to your local machine.
Install the required dependencies.
Make changes to the codebase and commit your changes using clear and concise commit messages.
Push your changes to your forked repository and create a pull request to merge your changes into the main branch of the project.

Testing
Before submitting a pull request, please ensure that everything works.

Reporting Issues
If you find a bug or have a suggestion for a new feature, please create an issue in our GitHub issue tracker. Please include a detailed description of the issue or feature request, as well as any relevant code or error messages.

Code of Conduct
We expect all contributors to adhere to our Code of Conduct. Please review the Code of Conduct in the file "CODE_OF_CONDUCT.md" before contributing to the project.

Contact Us
If you have any questions or concerns, please reach out to us at makarovdmitryd@yandex.ru. We appreciate your contributions and look forward to working with you!
